The directory structure for the project files are:

/			root of the project files
/rom			Common directory for DE1 and S3E. Contains the rom.vhd and tools to create rom files and fonts.
/doc			Documentation for both projects: DE1 and S3E
/DE1			Project files for Altera DE1 Development Board
/DE1/vhdl		vhdl files for Altera DE1 development board.
/DE1/memoryCores	memory cores specific to DE1
/S3E			Project files for Diligent Spartan 3E starter kit.
/S3E/vhdl		vhdl files for Diligent Spartan 3E development board.
/S3E/memoryCores	memory cores specific to S3E
